Chi sono

Sono un Software Developer con 5 anni di esperienza, con una forte specializzazione in Big Data e sistemi backend enterprise. Nel mio lavoro quotidiano progetto e mantengo pipeline dati distribuite, API REST robuste e servizi backend scalabili, principalmente in ambienti Java e Scala con Apache Spark.

Sono appassionato di tecnologia a 360°: dall'integrazione di modelli AI al DevOps con container e orchestrazione Kubernetes, fino alla formazione continua attraverso certificazioni professionali. Credo che imparare costantemente sia l'unico modo per restare rilevanti in un settore che cambia ogni giorno.

Nei progetti personali amo sperimentare stack diversi da quelli che uso in produzione — Python, Flask, strumenti AI — per espandere la mia visione e portare nuove idee anche nel lavoro enterprise.
