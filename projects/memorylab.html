MemoryLab

Python Flask JavaScript CSS Animations

Gioco di memoria web con tre livelli di difficoltà (Facile, Medio, Difficile) che varia il numero di coppie di carte. Le carte si girano con animazioni CSS 3D fluide, il punteggio tiene conto del tempo impiegato e delle mosse effettuate, e le statistiche di gioco vengono salvate in localStorage per una dashboard persistente tra le sessioni.

Funzionalità principali

  • Tre livelli di difficoltà — da 8 coppie (Facile) a 18 coppie (Difficile), con griglia adattiva
  • Animazione CSS 3D flip delle carte — rotazione realistica con transform: rotateY(180deg) e backface-visibility
  • Sistema di scoring basato su tempo e numero di mosse — più sei veloce, più punti guadagni
  • Dashboard statistiche persistente in localStorage — storico delle ultime partite, record personale
  • Flask serve l'applicazione e gestisce la logica di sessione lato server
  • Shuffle delle carte con algoritmo Fisher-Yates per garantire casualità uniforme

Stack tecnico

  • Python / Flask — server che gestisce le rotte e la logica di sessione
  • JavaScript (Vanilla) — logica di gioco, gestione degli eventi click, timer e calcolo punteggio
  • CSS Animations & 3D Transforms — flip delle carte, transizioni di stato, effetti hover
  • localStorage — persistenza dei dati statistici tra le sessioni senza backend database
