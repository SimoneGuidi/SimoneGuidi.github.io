FlowBoard

Python Flask Groq API JavaScript Drag & Drop API

Kanban board con drag & drop nativo HTML5 e un AI assistant contestuale integrato. L'assistente conosce lo stato corrente del board in tempo reale — colonne, card, loro posizioni — e può rispondere a domande, suggerire priorità o aiutare a organizzare il lavoro direttamente nel contesto delle attività presenti.

Funzionalità principali

  • Drag & drop nativo con HTML5 Drag and Drop API — nessuna libreria esterna, eventi dragstart, dragover, drop
  • Colonne Kanban configurabili: Todo, In Progress, Done — con conteggio card dinamico
  • AI assistant integrato (Llama3 via Groq API) che riceve lo stato corrente del board come contesto nel prompt
  • L'assistente può suggerire come organizzare le attività, identificare colli di bottiglia o rispondere a domande sul flusso di lavoro
  • Flask gestisce le rotte, lo stato del board e le chiamate all'API Groq lato server
  • Aggiunta, modifica e rimozione di card in ogni colonna con persistenza in sessione

Stack tecnico

  • Python / Flask — gestisce lo stato del board, serve la UI e orchesta le chiamate AI
  • HTML5 Drag and Drop API — implementazione drag & drop nativa senza jQuery UI o librerie di terze parti
  • Groq API + Llama3 — inferenza LLM a bassa latenza; il board corrente viene serializzato e iniettato nel system prompt
  • JavaScript (Vanilla) — gestione UI, eventi drag, aggiornamenti DOM e comunicazione con il backend Flask
