MoodMapper

Python Flask Groq API Llama3 Chart.js

Applicazione web di analisi del sentiment che utilizza Llama3 via Groq API per rilevare emozioni dominanti in un testo. Oltre all'analisi istantanea, mantiene una dashboard storica con la distribuzione emotiva nel tempo, permettendo di tracciare l'evoluzione del tono nei propri contenuti o nelle proprie note.

Funzionalità principali

  • Analisi del sentiment tramite LLM (Llama3) — identifica emozioni dominanti con sfumature oltre il semplice positivo/negativo
  • Integrazione Groq API per inferenza veloce — risposte in pochi secondi anche per testi lunghi
  • Dashboard storica con Chart.js — visualizzazione della distribuzione emotiva nel tempo tramite grafici a torta e a barre
  • Persistenza delle analisi in sessione — storico consultabile senza database esterno
  • UI minimalista con area di testo e feedback visivo immediato sull'emozione rilevata

Stack tecnico

  • Python / Flask — backend che orchestrala chiamata a Groq API e gestisce la logica di sessione
  • Groq API + Llama3 — modello LLM open-source usato per l'analisi semantica del testo con latenza molto bassa
  • Chart.js — grafici interattivi per la visualizzazione storica della distribuzione emotiva
  • Prompt engineering — istruzioni strutturate al modello per ottenere output JSON parsable con emozioni categorizzate
