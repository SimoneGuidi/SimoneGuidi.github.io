PyCalc

Python Flask JavaScript

Calcolatrice web con architettura client-server: la UI in JavaScript gestisce l'input e la visualizzazione, mentre Flask valuta le espressioni matematiche lato server in modo sicuro, senza mai usare eval(). Il risultato è un'applicazione leggera, sicura e con un'interfaccia dark theme curata.

Funzionalità principali

  • Valutazione sicura delle espressioni matematiche lato server — nessun rischio di code injection da eval()
  • Interfaccia dark theme responsive con tastierino numerico e operatori
  • Gestione errori: divisione per zero, espressioni malformate, overflow
  • Comunicazione asincrona tra frontend e Flask via fetch API
  • Deploy semplice con un singolo file Python — nessuna dipendenza esterna pesante

Stack tecnico

  • Python / Flask — server HTTP che espone un endpoint /calculate e valida/valuta l'espressione in modo sicuro
  • JavaScript (Vanilla) — gestione della UI, invio asincrono della richiesta e rendering del risultato
  • HTML & CSS — interfaccia dark minimalista con grid layout per il tastierino
