CryptoBoard

Python Flask Chart.js CoinGecko API

Dashboard crypto in tempo reale che aggrega dati di mercato tramite la CoinGecko API pubblica. Mostra prezzi aggiornati, variazioni percentuali e grafici storici a 7 giorni per le principali criptovalute. Include un sistema di caching lato server per rispettare i limiti di rate della API gratuita senza degradare l'esperienza utente.

Funzionalità principali

  • Prezzi in tempo reale aggiornati tramite polling — supporta BTC, ETH, e altre crypto principali
  • Grafici storici a 7 giorni renderizzati con Chart.js — visualizzazione andamento del prezzo
  • Sistema di caching lato server (Flask) per limitare le chiamate alla CoinGecko API e gestire il rate limiting
  • Variazioni percentuali a 24h con colori rosso/verde dinamici per una lettura immediata
  • Interfaccia dark theme responsive, leggibile sia su desktop che su mobile

Stack tecnico

  • Python / Flask — backend che gestisce le chiamate API, il caching e serve i dati al frontend
  • CoinGecko API — fonte dati gratuita per prezzi, variazioni e storico delle criptovalute
  • Chart.js — libreria JavaScript per la visualizzazione dei grafici storici a 7 giorni
  • Caching Flask — strategia TTL-based per ridurre le chiamate esterne e rispettare i rate limit
